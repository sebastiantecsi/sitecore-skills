This is a Claude Code skills repository for building Sitecore Marketplace apps.

It contains 7 skills in `.claude/skills/` that guide developers through scaffolding, building, and deploying Marketplace apps using the Sitecore Marketplace SDK (v0.4).

The skills target **Next.js App Router** apps deployed on **Vercel**, using the **Blok** design system (Sitecore's shadcn-based theme).

## Key files

- `.claude/skills/*/SKILL.md` — Skill definitions (frontmatter + instructions)
- `.claude/skills/*/references/*.md` — Pre-curated API docs and code patterns
- `README.md` — Skills catalog and install instructions
